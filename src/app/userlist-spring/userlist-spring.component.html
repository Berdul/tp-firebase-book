Liaison d'une API REST Spring à un front Angular

Le but de ce travail est de consommer une API Spring avec un front Angular. L'API joue le rôle d'interface permettant d'effectuer un lot de requêtes qui retourne des données au format JSON. Le front reçoit alors les données et permet de les afficher.
Techno utilisées :
  • Angular 8
  • Spring-boot
  • MySQL
  • Host API Heroku
  • Host Web Google

A terme, cette API sera le point central d'un jeu de Poker tour par tour. Couplée à un DB, elle effectuera toutes les requêtes nécessaires à la création d'un joueur,d'une partie multi-joueurs, au déroulement d'un tour de jeu et à l'utilisation d'un tableau des scores(Leaderboard).

Ici, la donnée reçu est une liste de joueurs enregistrés.
