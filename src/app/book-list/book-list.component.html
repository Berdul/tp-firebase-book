Bibliothèque en ligne

Liaison d'une DB Firebase à un front Angular

Cette bibliothèque est conçu pour stocker des données venant de l'utilisateur afin d'enregister les livres qu'il aime ou qu'il possède dans sa bibliothèque.
Techno utilisées :
  • Angular 8
  • Firebase
  • Host Web Google

Ici, la donnée reçu est une liste de livres au format Json.
